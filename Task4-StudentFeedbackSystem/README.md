# 🎓 Student Feedback Management System

A simple and user-friendly Flask web application that allows students to submit feedback and view submitted feedback records. This project demonstrates Flask routing, form handling, file operations, and responsive web design using Bootstrap.

## 🚀 Features

- 📝 Submit student feedback through a web form
- 👤 Capture student name and email
- 💬 Store feedback in a text file
- 📋 View all submitted feedback
- 🎨 Modern and responsive UI using Bootstrap
- ⚡ Lightweight and easy to understand

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Bootstrap 5

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/JyostnaMayeeSahu079/ALFIDO-TECH.git
```

### 2. Navigate to Project Folder

```bash
cd Task4-StudentFeedbackSystem
```

### 3. Install Flask

```bash
pip install flask
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```text
http://127.0.0.1:5000
```

## 📸 Application Workflow

1. Student enters Name, Email, and Feedback.
2. Feedback is submitted through the form.
3. Data is stored in `feedback.txt`.
4. User is redirected to the feedback list page.
5. All submitted feedback can be viewed.
