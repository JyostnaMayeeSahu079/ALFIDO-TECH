# 🌐 Task 2 - API Integration & JSON Handling

## 📌 Project Overview

This project was developed as part of the **ALFIDO TECH Python Development Internship**.

The application fetches real-time weather information from a public API and displays the results in a user-friendly format. It demonstrates API integration, JSON parsing, search functionality, and exception handling using Python.

---

## 🎯 Objectives

* Learn API integration using Python
* Handle JSON data responses
* Implement search functionality
* Manage API and connection errors gracefully

---

## ✨ Features

✔️ Fetch real-time weather information

✔️ Search weather by city name

✔️ Parse JSON responses

✔️ Display temperature, humidity, weather conditions, and wind speed

✔️ Handle connection and API errors

✔️ Allow multiple city searches in a single session
