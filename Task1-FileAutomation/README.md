# 📂 Task-1 : Python File Handling & Automation

## 🚀 Overview

This project was developed as part of the **ALFIDO TECH Python Internship Program**.

The application is a menu-driven **File Automation System** that demonstrates Python file handling, automation, and exception handling concepts. It enables users to perform common file management operations efficiently through a simple command-line interface.

---

## ✨ Features

✔️ Create Files

✔️ Read File Contents

✔️ Rename Files

✔️ Move Files to a Backup Directory

✔️ Delete Files

✔️ Exception Handling (Try-Except)

✔️ Menu-Driven User Interface

---
